# Security Policy

## Supported Versions

Only the latest version receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you find a security vulnerability, please do not open a public issue.

Report it via the [Discussions](../../discussions) tab using the **Security** category. Describe the vulnerability, how to reproduce it, and the potential impact.

You can expect an acknowledgment within a few days. If the vulnerability is confirmed, a fix will be prioritized and you will be updated as it progresses. If it is declined, you will get an explanation why.
